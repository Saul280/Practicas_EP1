Pantalla OLED

Saúl Adair Jarquin Ramos
Carrera en Ingeniería Biomédica
Fecha: Septiembre 30, 2023
Arduino Elementos programables

- Resumen -

Se presenta la pantalla OLED por I2C, en donde se configura el Attiny85 para imprimir información en display, usando como ejemplo el estado de un botón, ya sea que este apagado o prendido.

- Introducción -

Mediante una simulación previamente hecha en la página WOWKI, se escribirá un código y simulará un circuito en el cual estén conectados un Attiny85 y una pantalla OLED la cual mostrará el estado "Low" o "High" dependiendo de si el botón (input) este presionado o no.

- Materiales -

-Arduino UNO

-ATTINY85

-Pantalla OLED

-Protoboard

-Resistencias

-Botones push

-Cables

- Desarrollo -

Se utilizó la página web WOWKI para hacer pruebas de simulación, ya que se encuentran varias herramientas útiles para esta práctica, como un compilador de código adecuado para programar Attiny85, además de incluir una simulación gráfica para hacer conexiones entre varios componetes del cicuito.

- Conclusiones -

La pantalla OLED muestra un estado "High" al principio para que al estar presionado el botón pasara a un estado Low, esto debido a las conexiones hechas por el botón que no son las convencionales, puesto que esta conectado a negativo y deja pasar la corriente negativa a la entrada para entrar en estado Low, por lo que se puede decir que la entrada está en High todo el tiempo.
