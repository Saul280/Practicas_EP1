Proyecto dispositivo werable

Saúl Adair Jarquin Ramos
Carrera en Ingeniería Biomédica
Fecha: Octubre 27, 2023
Arduino Elementos programables

- Resumen -

Se diseña un prototipo de un dispositivo tipo werable, el cual tiene como cunción detectar diferentes señales por medio de distintos sensores, además de tener una interfaz interactiva que permite elegir la función que el usuario desee.

- Introducción -

Se presenta un prototipo en protoboard con dos sensores (uno de temperatura y otro de imanes), que fungen como un preludio del objetivo final que será diseñar un PBC para hacer un dispositivo tipo werable.

A continuación se muestra el ejemplo del prototipo hecho ya en protoboard y se deja el texto del codigo utilizado para programar el Attiny85 para descargarlo.

- Materiales -

-Arduino UNO

-ATTINY85

-Pantalla OLED

-Sensor de tempersatura

-Sensor de imanes

-Protoboard

-Resistencias

-Botones push

-Cables

- Desarrollo -

Se utilizó la página web WOWKI para hacer pruebas de simulación, ya que se encuentran varias herramientas útiles para esta práctica, como un compilador de código adecuado para programar Attiny85, además de incluir una simulación gráfica para hacer conexiones entre varios componetes del cicuito.

Código

Se puede observar el código final utilizado para el dispositivo Werable, se puede observar en el archivo de texto en la parte de descargables.

Diseño

Se crea un diseño en un progrma para diseñar placas y se generan los archivos a enviar para su posterior impresión, dichos archivos se encuentran en la parte de descargables.

Construcción

La construcción de este circuito en un principio se hará en un protoboard, con las conexiones hechas mediante cables y como fuente se usará la energía transmitida por un Arduino UNO conectado.

Posteriormente se realiza un placa impresa del circuito y una vez hecha, se soldan los componetes y se alimente con un a fuente de alimentación, en este caso una batería de 4.8V, para luego observar y comprobar su funcionamiento.

Página web

Por último, se diseño un instructable como pagina web oficial donde se encuentra de manera más extensa y explicada el desarrollo y funcionamiento del werable.

https://www.instructables.com/Design-About-a-Hand-Watch-That-Evaluates-the-Envir/
